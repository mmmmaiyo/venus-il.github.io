Which of the following is the correct CSS syntax for padding?

  • padding: -10px 0px;

  • padding-right: 5px;

  • padding: 10px 15px 5px 7px 22px;

  • padding-down: 15px;

So close!

An element's padding is the space between its content and its border. This property can have from one to four values. However, negative values are not allowed

See https://www.w3schools.com/cssref/pr_padding.asp for more information.

How should you configure your table width such that it adjusts to the width of the current window?

  • width: 100%;

  • width: 1960px;

  • width: full-screen;

  • width: all;

That's not right.

The width attribute specifies the width of the element, in pixels, and does not include padding, borders, or margins. A % value defines the element's width as a percentage of its containing block.

See https://www.w3schools.com/cssref/pr_dim_width.asp for more information.
